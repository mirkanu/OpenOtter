## Bugs Found and Fixed During UAT

1. **Missing `filename` in transcribe request** — AudioUploader.tsx sent only `filepath` to `/api/transcribe`; Zod schema required both. Fixed: added `filename: uploadData.filename` to request body.
2. **ASSEMBLYAI_API_KEY not in process env** — PM2 `env_file` directive was not propagating the key. Fixed: added explicitly to `env` block in ecosystem.config.js.
3. **DB save never triggered** — transcript page navigated to `/transcript/[jobId]` without `filename`/`filepath` URL params; poll requests had no query params so `saveRecording()` guard `if (filename && filepath)` always failed. Fixed: params now threaded through URL and polling requests.
4. **Duration showing 0:00** — `formatDuration` in RecordingCard.tsx divided by 1000 treating seconds as milliseconds. AssemblyAI `audio_duration` is in seconds. Fixed: removed the `/1000` divisor.
