## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge padding, inline gaps |
| sm | 8px | Card internal padding, metadata item gaps |
| md | 16px | Page horizontal padding (`px-4`), card vertical padding |
| lg | 24px | Section vertical spacing, gap between cards |
| xl | 32px | Header height clearance, top-of-list padding |
| 2xl | 48px | Empty state vertical centering offset |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Touch targets (header button, card tap area): minimum 44px height per iOS HIG
- Header height: 56px (fits 44px touch target with 6px vertical padding each side)
- Container: `max-w-2xl mx-auto px-4` — carries over from existing pattern (source: CONTEXT.md D-08)

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label / Metadata | 14px | 400 (regular) | 1.4 |
| Heading (card title) | 16px | 600 (semibold) | 1.3 |
| Display (page section heading) | 20px | 600 (semibold) | 1.2 |

**Font weights in use:** 400 and 600 only.

**Truncation rule:** Recording titles on cards truncate at one line (`truncate` class). Titles on the detail page wrap fully.

## Color

shadcn neutral palette (CSS variable strategy, dark mode via `next-themes` class strategy).

| Role | Light Mode Value | Dark Mode Value | Usage |
|------|-----------------|-----------------|-------|
| Dominant (60%) | `--background` (white, hsl 0 0% 100%) | `--background` (hsl 0 0% 3.9%) | Page background, main surface |
| Secondary (30%) | `--card` (white with border) | `--card` (hsl 0 0% 7.8%) | Recording cards, sticky header background |
| Accent (10%) | `--primary` (hsl 0 0% 9%) | `--primary` (hsl 0 0% 98%) | Reserved for: "New Transcript" button only |
| Destructive | `--destructive` (hsl 0 84.2% 60.2%) | `--destructive` (hsl 0 62.8% 30.6%) | Not used in this phase (no delete actions) |

Accent reserved for: the "New Transcript" button in the sticky header and the empty state CTA button. No other interactive elements use the primary/accent color — links and card hovers use `--muted` / `hover:bg-accent` from shadcn defaults.

**Status badge colors (semantic — separate from the 60/30/10 palette):**
- `completed`: `bg-green-100 text-green-800` (light) / shadcn success variant
- `error`: `bg-red-100 text-red-800` (destructive variant)
- `processing` / `pending`: `bg-yellow-100 text-yellow-800` (warning, no shadcn default — use inline classes)

## Interaction Contracts

### Recording Card
- Tap target: full card area, minimum 44px tall
- Hover: `hover:bg-accent` (shadcn default card hover)
- Active/press: `active:scale-[0.98]` for tactile iOS feedback
- Transition: `transition-colors duration-150`
- Status badge: top-right of card, always visible

### "New Transcript" Button (header)
- Position: sticky header, right-aligned
- Size: `h-9` (36px) — acceptable at header context; wraps in 56px header for 44px touch area
- Visual: shadcn `Button` default variant (primary/filled)
- Opens: upload modal or navigates to `/upload` (implementation decision for executor)
- Pending state: show `Loader2` lucide icon spinning while upload is in progress

### Export Button (detail page)
- Position: below metadata block, above transcript
- Visual: shadcn `Button` variant `outline`
- Icon: `Download` from lucide-react, 16px, left of label
- Behavior: clipboard if content ≤ 50KB, else triggers `.md` file download (source: CONTEXT.md D-13)
- Toast: use `sonner` — success message per copywriting contract above
- Disabled state while export is computing

### Skeleton Loading
- Recording cards: show 6 skeleton cards on list load
- Skeleton card: full card shape — one wide bar (title) + two narrow bars (metadata)
- Detail page: header skeleton (4 metadata bars) + transcript skeleton (10 alternating-width bars)

### Dark Mode Toggle
- Location: sticky header, left of "New Transcript" button
- Icon: `Sun` / `Moon` from lucide-react
- Uses `next-themes` `useTheme` hook
- Renders `null` on server (avoid hydration mismatch with `mounted` guard)

```

  "OpenOtter" (link /`)  ← left, font-semibold text-lg
  <DarkModeToggle />      ← ghost icon button
  <Button>"New Transcript"</Button>  ← right
</StickyHeader>

<main className="pt-[56px]">   ← clears sticky header
  <div className="max-w-2xl mx-auto px-4">
    {page content}
  </div>
</main>
```

```

  </section>

```
